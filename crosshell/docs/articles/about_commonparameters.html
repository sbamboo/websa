Common Parameters

about_CommonParameters

Common parameters are parameters handled by the shell, and work for every command (that haven't disabled them).

They exists to give global functions across crosshell, like show information and likewise.


CommonParameters:

Help:

The help parameter (/help) runs the command through the get-help cmdlet. And shows information about a commmand and it's parameters.

/? -? and /h are aliases for the help parameter.
Search:

Searches for the command specified through the help cmdlet.


Webi:

Runns the command through webi.


Calc:

Runns the command through calc.
