Crosshell Scripting (Zedix)

about_crosshellscripting

Crosshell has some scripting functionalty built in with the Interpriter but also with the script command.


The script command allowes you to run .crcmd files or crosshellCommand/crosshellScript file.


Crosshell Scriptfile

The .crcmd scriptfile contains crosshell commmands just like in the console and is run one after the other.

There is also support for commands like while and if.


There also is an extended file mode but it is experimental (To enable write '# crosshell.exprMulti:true' at the top of the file)

To run a script file in crosshell run:

Script -file "filename"
