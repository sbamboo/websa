BuiltIn Commands

about_builtincommands

BuiltIn commands in crosshell are commands shipped with shell or installed on first start. These contain core features to crosshell and other necessities.

BuiltIn packages can be recognised by beginning with "crosshell_".


(Note some packages beginning with "crosshell_" are not builtins)

To list crosshell packages run

PackageHand -Search "Crosshell_*"
